# paint_canvas_js

Painting board made with Vanilla Js  
https://narae-tech.github.io/paint_canvas_js/

<br>

## 프로젝트 주제

- 캔버스에 그림을 그리고 저장할 수 있는 웹 페이지 제작

<br>

## 프로젝트 참여 인원

- 1인(토이 프로젝트)

<br>

## 프로젝트 기간

- 2021.10.12 ~ 2021.10.12

<br>

## 개요

- vanilla javascript 학습을 위한 프로젝트로 캔버스위에 그림을 그리고, 배경색을 칠하고 그림을 저장할 수 있는 웹 페이지 제작을 목표로 하였다.

<br>

## 프로젝트 구성 안내(기술 스택)

- html5
- css3
- javascript(vanilla js)

<br>

## 느낀점

- Canvas API 사용법 터득
  - 애니메이션, 게임 그래픽, 데이터 시각화, 사진 조작 및 실시간 비디오 처리를 위해 사용되는 canvas API 사용법 터득
- javascript 사용 능력 향상
